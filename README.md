## Setting up the Environment

1. **Create a new virtual environment:**

```bash
conda create -p venv .venv python=3.11.7
```

2. **Activate the virtual environment:**

```bash
conda activate .venv
```

3. **Install required packages:**

```bash
pip install -r requirements.txt
python==3.11.7 -y
```
